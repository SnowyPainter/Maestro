>

> 브랜드의 리듬을 잃지 않은 채로.

## ⚙️ **Section 2 — What is Maestro**

### **Headline**

> 브랜드를 지휘하는 지능형 오케스트라

### **Body**

> Maestro는 단순한 자동화 툴이 아닙니다.
>
> **당신의 판단과 행동을 하나의 시스템으로 연결하는 엔진**입니다.
>
> 모든 발행, 수정, 분석이 하나의 지휘 아래 작동하며,
> 매번의 결과가 다음 행동의 근거가 됩니다.

### **Visual**

흐르는 그래프 형태로:
**Persona → Playbook → Orchestrator → CoWorker**
순환하는 루프 애니메이션

---

## 🎹 **Section 3 — Core Components**

### **Headline**

> 4개의 엔진이 당신의 판단을 지휘합니다.

| 구성요소                | 설명                         | 키워드                                |
| ------------------- | -------------------------- | ---------------------------------- |
| 🎭 **Persona**      | 브랜드의 목소리, 말투, 금칙어를 기억하는 자아 | *Voice, Tone, Identity*            |
| 📖 **Playbook**     | 행동과 결과를 학습해 인사이트로 남기는 기억   | *Learning, Memory, Feedback*       |
| 🧭 **Orchestrator** | 모든 행위를 결정론적으로 실행하는 지휘자     | *Control, Trace, Determinism*      |
| 🤝 **CoWorker**     | 당신이 부재할 때도 루틴을 이어가는 동료     | *Automation, Continuity, Presence* |

**Subline:**

> 인간은 판단하고, 시스템은 그 판단을 반복합니다.

## 🧠 **Section 5 — How it works (Flow)**

### **Headline**

> 한 번의 행동이 루틴이 됩니다.

1️⃣ **감지** — Sniffer가 시장의 신호를 탐지
2️⃣ **생성** — Persona와 Playbook이 지침을 구성
3️⃣ **실행** — Orchestrator가 행동을 결정
4️⃣ **학습** — CoWorker가 결과를 기록

**Subline:**

> 이 루프는 끊임없이 당신의 판단을 복제하고 정제합니다.

# ✍️ **Tone & Style Guide (for Copywriting)**

| 요소    | 스타일                         | 예시                                              |
| ----- | --------------------------- | ----------------------------------------------- |
| 문장 길이 | 짧고 단정 — 쉼표보단 줄바꿈            | “당신의 판단을 복제하는 중입니다.”                            |
| 어휘    | 기술적 단어와 시적 표현 병치            | “결정론적 지휘”, “기억하는 엔진”                            |
| 시각 리듬 | 음악적. 3단 구조 / 쉼              | “한 번의 행동이 기억이 되고 / 그 기억이 판단이 되고 / 판단이 브랜드가 된다.” |
| 색감    | 검정/백 + 단일 포인트컬러 (코발트 or 골드) | -                                               |

# 🧩 요약: 서사적 흐름

> **0. 선언** — 복제의 시작
> **1. 문제** — 반복되는 판단의 피로
> **2. 정의** — 2단계 자동화 (Draft → Schedule)
> **3. 구조** — 4개의 지휘 엔진
> **4. 체험** — "Get trends and create draft" → Schedule
> **5. 작동 원리** — 트렌드 수집 → LLM 생성 → 정시 발행
> **6. 검증** — 데이터로 기억 강화
> **7. 철학** — 판단의 복제, 리듬의 윤리
> **8. 증언** — 인간의 목소리
> **9. 행동 유도** — "당신의 판단을 복제하세요."
