# Persona의 역할 정의

## 개념

* Persona는 브랜드 목소리/톤/규칙/제약을 표현하는 DTO
* 모든 생성 / 실행 / 검증 단게에서 공통 컨텍스트로 주입되어 브랜드 일관성과 자율성을 동시에 보장한다.

## 3가지 관점

* 브랜드 규범의 보관소:
    - tone, style, 금칙어, 해시태그, 링크 규칙 등 브랜드의 일관성을 유지하는 데이터 원천
    - Voice
    - Rule
    - Identity
* 생성 컨텍스트:
    - Adapter가 모든 행동을 할 때 참조하는 상위 컨텍스트
    - Context Injection
* 성과 단위:
    - 장기적인 하위 Campaign들의 집계 단위
